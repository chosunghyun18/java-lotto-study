## 기능 요구사항 정리

### Player
- [x] Lotto 구입 금액 입력
  - [x] 1,000원으로 나누어 떨어지지 않는 경우 예외 처리
- [x] 당첨 번호 입력
  - [x] 쉼표로 구분
  - [ ] validation 판별
- [ ] 보너스 번호 입력
  - [ ] 당첨 번호와 중복이 안되는지
  - [ ] 1~45 사이 정수 인지
  - [x] 숫자가 맞는지

### Lotto
- [ ] 로또 번호의 validate 판별
  - [x] 6자리
  - [ ] 중복 안됨
  - [ ] 1~45 사이의 정수
  - [x] 숫자가 맞는지

### Lotto Machine
- [x] 발행한 로또 수량 출력
- [x] 로또 수량에 맞게 로또 번호 6개 출력
  - [x] 오름차순으로 정렬
  - [x] 1~45 사이 정수
  - [x] 중복 없음

### Result Machine
- [ ] 몇개 일치하는지 계산 및 출력
- [ ] 수익률 계산
