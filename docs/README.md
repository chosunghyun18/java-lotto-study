## 기능 요구사항 정리

### Player 
- [ ] Lotto 구입 금액 입력 
  - [ ] 1,000원으로 나누어 떨어지지 않는 경우 예외 처리
- [ ] 당첨 번호 입력
  - [ ] 쉼표로 구분
  - [ ] 중복 되지 않아야 함
  - [ ] 1~45 사이 정수
  - [ ] 숫자가 맞는지 확인
- [ ] 보너스 번호 입력
  - [ ] 당첨 번호와 중복이 안된는지
  - [ ] 1~45 사이 정수
  - [ ] 숫자가 맞는지 확인 

### Lotto Machine
- [ ] 발행한 로또 수량 출력
- [ ] 로또 수량에 맞게 로또 번호 출력
  - [ ] 오름차순으로 정렬
  - [ ] 1~45 사이 정수
  - [ ] 중복 없음

### Result Machine
- [ ] 몇개 일치하는지 계산 및 출력
- [ ] 수익률 계산
